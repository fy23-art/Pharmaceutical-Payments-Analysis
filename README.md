# Pharmaceutical-Payments-Analysis
Pharmaceutical Payments and Physician Prescribing Behavior in the United States

- Boosted model fit by 32% using log transformations on 1.2M+ pharmaceutical payment records and reduced computation time by 60% through optimized SQL preprocessing of 20M+ rows. 
- Produced correlation heatmaps, killer plots, and LOESS smoothing to assess physician prescribing behavior.
